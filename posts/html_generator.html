html template generator using C language

I am maintaining this blog by posting my devlogs.
My writing process is as follows:

  1. write a content of a post using a software called "ghostwriter" (a markdown editor)
  2. convert the markdown file to a html file using "ghostwriter" Ghostwriter
  3. write html a head, meta and style tags to make a complete html5 file containing my writing.
As I have been writing several posts in the last few weeks,
I realized that they all share more or less similar html format.
Espeically, the first part of the html, the style and the head tags,
and the last html part (all those closing tags) after the content are almost always repetitive.
Because I use mostly the same style (background and font) throughout all posts,
I figured that I might as well make a little program to generate
a html template to wrap the content of the post.
This C program takes the content as input, and wraps that content with the html head part and closing tags.
Therefore, I do not need to write the same html meta and style tags every time I write the post. My program will do it for me.
This html post itself is in fact generated by the program.
