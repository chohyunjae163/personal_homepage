Configuring 4coder!

I have been using a text editor called 4coder for my project lately.
IDEs like Visual Studio is certainly very convinient and has a lot of features.
However, because it does a lot of work (which is sometimes repetitive and bothersome)
under the hood, a novice programmer like me cannot learn
and experience the process of compilation and linking.
Thus, I like to use a text editor and a separate debugger instead and compile and debug myself.
4coder has been my favourite text editor so far, simple, minimalistic and performant.
4coder
Also, 4coder is very customizable like emacs.
Because this editor can be programmed with its customization API in C/C++ ,
I can read the customization layer code and alter the editor to my liking, although recompilation is necessary.
Working on my cave-hunter project today, I added a custom command for "auto-close parenthesis".
If I press a Shift+9, the editor adds two parenthesis and a space between.
This is a very tiny feature, but I needed this and felt great implementing a new custom command
to my favourite text editor.


//4coder_combined_write_commands.cpp
CUSTOM_COMMAND_SIG(open_long_parens)
CUSTOM_DOC("At the cusor, insert a '(' and ')' separated by a space.")
{
    char text[] = "(  )";
    i32 size = sizeof(text) - 1;
    long_braces(app, text, size);	
}
//4coder_default_map.cpp
Bind(open_long_parens,      	 KeyCode_9, KeyCode_Shift);
