Recently, I learned about c language alternatives, zig and odin. Both of them look fascinating. I first heard about zig on Youtube a few years ago, clip. It was fascinating. I did not know there are people who develop low-level, system programing languages these days ("modern languages" that I heard were all the js languages or "high level" languges). Because I was interested in low-level programming, working closely with operating systems and hardwares ,the talk about this alternative C language seemed very attractive.
Odin, I found about that language while reading gingerBill's interesting articles. gingerBill. I loved to read his articles about programming and got interested in a language called odin on which he is working. I decided to give odin a try for my next toy project (minesweeper clone) because it has full support for all the major graphics API such as Vulkan, OpenGL, Direct3D, and Metal.
Also, the language is said to be "data-oriented" and the author says one of the principles of the language is "joy of programming" about which I care the most. Nonetheless, compare to to other popular languages, there is not much resource so I have to dig the language deeply by myself.
